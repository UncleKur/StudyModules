По способу организации данных коллекции деляться на несколько видов:
    Array - массив, структура данных фиксированного размера, в памяти которой друг за другом хранятся элементы одного типа.
Остальные коллекции основываются на массиве.
-Доступ к элементу по индексу
-Нельзя изменить размер массива
-Можно изменять элементы массива
    List - список, структура данных переменного размера, в памяти которой друг за другом хранятся элементы одного?
(или разных?) типа. В отличии от массива размер списка может быть изменен (увеличиваться или уменьшаться).
Может быть изменяемым или неизменяемым (mutable and immutable соотвественно)
    Set - множество
    Map - ассоциативный массив/представление/словарь

    LinkedList - список, в котором каждый элемент имеет ссылки на предыдущий и следующий элементы.
-Операции добавления и удаление просходят быстрее
-Получение и изменение элементов происходят медленее
    Queue - очередь, структура данных, работающая по принципу First in - First Out (первый вошёл - первый вышел)
Элементы добавляются в конец очереди, а извлекаются из её начала.
    Stack - структура данных, работающая по принципу First in - Last Out (последний вошёл - первый вышел)
